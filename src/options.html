Obsidian Clipper

An unoffical clipper

This is an unofficial clipper for Obsidian that allows you to easily clip a selection to a note in Obsidian. Made by Joost Plattel. This plugin is available as open-source on Github.

At this page you can configure the vault & note to append the clip to. Please test the configuration

Use {title}, {date}, {time}, {datetime}, {year}, {month}, {day}, {timezone}, and {zettel} as placeholders
Use {title}, {clip}, {blockquote}, {date}, {time}, {datetime}, {year}, {month}, {day}, {timezone}, {zettel}, {og:image} and {url} as placeholders
