Obsidian Clipper

An unoffical clipper for Chrome

This is an unofficial clipper for Obsidian that allows you to easily clip a selection to a note in Obsidian. Made by Joost Plattel. This plugin is available as open-source on Github. At this page you can configure the vault & note to append the clip to.

Use {date}, {datetime}, {title}, {zettel} and {url} as placeholders
